# 🦄 Unicorn Easter Egg for TYPO3
A small and delightful TYPO3 backend extension that brings a touch of magic to the daily editorial routine. This repository contains a hidden easter egg for all unicorn lovers.

## ✨ Features
Once the extension is installed, a hidden feature can be triggered within the TYPO3 backend:
* Trigger: Press the keyboard shortcut Command + U (or Ctrl + U on Windows).
* Effect: A small unicorn starts running across your screen.
* Interaction: Use the Arrow Keys to make the unicorn jump!

## 🚀 Compatibility & Installation
This extension is fully compatible with TYPO3 v14.

Clone the repository into your typo3conf/ext/ directory or install it via Composer:

Bash : `composer require luisasofie/unicorn`

Activate the extension in the Extension Manager or via CLI.

Reload the backend and enjoy the ride!

## 🛠 Technical Details
The extension uses lightweight JavaScript to listen for keyboard events and render animations within the backend viewport. It is designed to be non-intrusive and does not affect the performance of the core system.

Have fun jumping! 🌈
